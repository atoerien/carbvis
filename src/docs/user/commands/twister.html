Command: twister

Usage:
twister [ atom-spec ] [ replace  true | false ] [ update  true | false ] [ startEndCentroid  true | false ] [ ribSteps  N ] [ maxRingSize  N ] [ maxPathLen  length ] [ ribWidth  width ] [ ribWeight  height ] [ colorTop  linkage-colormap ] [ colorBottom  linkage-colormap ] [ gumTwist  true | false ]

The twister command creates a Twister model for the specified atoms. Each glycosidic linkage is represented by a flat ribbon that twists between the two rings it connects, aligning with the ring planes at each end. The top and bottom sides of each ribbon are colored independently (white and blue by default), showing the relative orientation of successive rings and the degree of backbone twist.

Twister emphasizes backbone twist and the relative orientation of rings. For ring puckering and absolute ring orientation, see paperchain. For overall backbone shape and linkage torsion-angle coloring, see strand.

If atom-spec is omitted, all atoms in all open atomic structures are used.

See also: CarbVis, paperchain, strand

replace  true | false
Whether to remove existing PaperChain models that overlap the new one (default true).
update  true | false
Whether to automatically update the model if the structure changes (default true).
startEndCentroid  true | false
When true (default), successive ribbons connect at the centroid of each ring, forming one smooth continuous ribbon that spans the whole molecule. When false, each ribbon segment starts and ends at the atoms involved in the glycosidic linkage rather than at the ring centroid.
ribSteps  N
The number of subdivision steps used when rendering each segment of the ribbon (default 10). Higher values result in a smoother ribbon curve.
maxRingSize  N
Maximum ring size considered when identifying carbohydrate rings (default 10 atoms).
maxPathLen  length
The maximum path length in bonds to search when finding linkages between rings (default 5).
ribWidth  width
The width of the ribbon in Å (default 0.3).
ribHeight  height
The thickness of the ribbon in Å (default 0.05).
colorTop  linkage-colormap
The linkage colormap used to color the top side of each ribbon (default blue).
colorBottom  linkage-colormap
The linkage colormap used to color the bottom side of each ribbon (default white).
gumTwist  true | false
Enables the "Twister Gum" variant (default false). The ribbon is thinned towards its center with a degree proportional to the total angular twist of that segment, and the twist rate is concentrated in the middle of each segment rather than being uniformly distributed along its length.
