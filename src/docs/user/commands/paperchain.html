Command: paperchain

Usage:
paperchain [ atom-spec ] [ replace  true | false ] [ update  true | false ] [ bipyramidHeight  height ] [ maxRingSize  N ] [ color  ring-colormap ] [ texFormula  stripes | grid | diamond | rings | waves ] [ texPeriod  pixels ] [ texDuty  fraction ]

The paperchain command creates a PaperChain model for the specified atoms. It identifies sugar rings within the specified atom-spec and renders each ring as a bipyramid whose color by default encodes the Cremer-Pople puckering amplitude of that ring, using a hot-to-cold color scale so that unusually puckered (high-energy) rings stand out clearly.

With texFormula, a texture is added to the "bottom" face of each ring polygon, additionally showing ring orientation.

PaperChain emphasizes ring puckering twist and absolute ring orientation. For overall backbone shape and linkage torsion-angle coloring, see strand. For backbone twist and relative ring orientation, see twister.

If atom-spec is omitted, all atoms in all open atomic structures are used.

See also: CarbVis, twister, strand,

replace  true | false
Whether to remove existing PaperChain models that overlap the new one (default true).
update  true | false
Whether to automatically update the model if the structure changes (default true).
bipyramidHeight  height
Height of the bipyramid apex above (and below) the ring plane, in Å (default 1.0).
maxRingSize  N
Maximum ring size considered when identifying carbohydrate rings (default 10 atoms).
color  ring-colormap
The ring colormap used to color the polygons. Defaults to the paperchain ring colormap, which maps the Cremer-Pople puckering amplitude of each ring to a hot-to-cold color scale, with colors specifically chosen so that the different principal ring conformers can be easily distinguished.
texFormula  stripes | grid | diamond | rings | waves
Applies a procedural texture to the bottom face of each ring polygon. Available patterns are waves (parallel sinusoids), stripes (horizontal lines), grid (horizontal and vertical lines), diamond (concentric diamond shapes), and rings (concentric rings).
When omitted, no texture is applied.
texPeriod  pixels
Pixel distance between repeating texture features (default 128).
texDuty  fraction
Fractional duty cycle of the repeating texture pattern, in the range [0, 1] (default 0.5).
