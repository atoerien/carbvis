Command: color bydihedral

Usage:
color bydihedral [ bond-spec ] [ color  linkage-colormap ] [ maxRingSize  N ] [ maxPathLen  length ]

The color bydihedral command colors bonds along glycosidic linkages according to the linkage's dihedral angles.
Note that halfbond mode will necessarily be disabled on all bonds colored (i.e. those that form part of a glycosidic linkage).

If bond-spec is omitted, all bonds in all open atomic structures are used.

See also: CarbVis, color, strand, twister

color  linkage-colormap
The linkage colormap used to color the bonds. Defaults to the default colormap.
maxRingSize  N
Maximum ring size considered when identifying carbohydrate rings (default 10 atoms).
maxPathLen  length
The maximum path length in bonds to search when finding linkages between rings (default 5).
