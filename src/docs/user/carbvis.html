CarbVis - Carbohydrate Visualizations for ChimeraX

CarbVis adds carbohydrate-specific visualizations to ChimeraX. The visualizations identify sugar rings and the glycosidic linkages between them, then render abstract 3D representations that highlight backbone shape, ring orientation, puckering, and linkage torsion angles.

Commands

  • paperchain - create a PaperChain model that renders each sugar ring as a bipyramid colored by ring puckering amplitude, with an optional texture to show ring orientation
  • twister - create a Twister model that draws a ribbon through each glycosidic linkage, twisting to align with the ring planes
  • strand - create a Strand model that draws a smooth tube through the sugar rings colored by dihedral angles
  • color bydihedral - color glycosidic linkage bonds by their dihedral angles

Linkage Colormaps

The strand, twister, and color bydihedral commands accept a colormap to color linkages. A constant color-spec may also be given instead of a colormap to color all linkages uniformly. Each colormap name may be followed by optional keyword arguments in parentheses.

  • default
    Colors each linkage white-to-red by how closely its dihedral angles match the expected ranges for its anomeric type (α or β). Linkages within all expected ranges are white; those outside are red.

    The expected ranges are:

    • For two-bond linkages (φ and ψ):
      • α linkages: φ ∈ (−70°, −10°); β linkages: φ ∈ (10°, 70°)
      • ψ ∈ (−45°, 45°)
    • For three-bond linkages (φ, ψ, and ω):
      • α linkages: φ ∈ (−70°, −10°); β linkages: φ ∈ (10°, 70°)
      • ψ near ±60° or ±180°
      • ω near ±60° or ±180°

    Range edges are smoothed with super-Gaussian functions.

    palette  palette
    A ChimeraX color palette. Default is a white-to-red ramp.

    Examples:

    strand color default
    strand color default(palette="purples")

  • simple
    Colors each two-bond linkage by how well φ and ψ fall within a user-defined rectangular window. Linkages inside the window are white; those outside are red.

    Range edges are smoothed with super-Gaussian functions.

    phi_min  degrees
    Lower bound for φ (default -60).
    phi_max  degrees
    Upper bound for φ (default 60).
    psi_min  degrees
    Lower bound for ψ (default -60).
    psi_max  degrees
    Upper bound for ψ (default 60).
    palette  palette
    A ChimeraX color palette. Default is a white-to-red ramp.

    Examples:

    strand color simple(phi_min=-40, phi_max=40)

  • norm
    Colors each linkage by the Euclidean norm of its dihedral angle vector, normalised to [0, 1].

    palette  palette
    A ChimeraX color palette. Default is a white-to-red ramp.

    Examples:

    color bydihedral color norm(palette=blues)

Ring Colormaps

The paperchain command, and the sphereColor option of strand, accept a ring colormap to color rings, or a constant color-spec to color all rings uniformly.

  • paperchain
    Colors each ring based on its Cremer-Pople puckering amplitude using a hot-to-cold color scale. No arguments are accepted.
